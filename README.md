# Proyecto React + Base de Datos

Aplicación web hecha con React y Node.js que permite el inicio de sesión de usuarios conectado a una base de datos. Incluye validación de credenciales, manejo de rutas y una interfaz con diseño personalizado en tonos negro y dorado.
